## Entity & Relation Patterns Observed

This source class (procedural / experimental knowledge) is structurally different from the
claim-centric content of research articles. Instead of asserting biological *facts* ("gene X
causes disease Y"), these sources encode **how to produce, measure, or transform** biomedical
material. The dominant relationships are **procedural and operational** rather than causal-biological.
Both flavors matter for BioOKF: the procedural graph is what lets the KG represent *provenance,
methods, and reproducibility*, while the embedded biological entities (genes, organisms, cell
types, compounds) overlap heavily with the article-derived ontology.

### Recurring ENTITY (node) types

**Biological / biomedical entities (overlap with article ontology):**
- **Organism / model system** ‚Äî E. coli, S. cerevisiae (yeast), mouse, zebrafish, Drosophila,
  C. elegans, Arabidopsis, human; pathogens (M. tuberculosis, SARS-CoV-2).
- **Gene / locus / gene region** ‚Äî target genes for KO/KD, 16S rRNA gene, V3-V4 region, sgRNA targets.
- **Variant / mutation** ‚Äî SNVs, indels, somatic mutations, CRISPR-induced frameshifts, lineages/variants.
- **Protein** ‚Äî blot targets, purified recombinant proteins, antibody antigens, Œ≥H2AX, GroEL.
- **Cell type / cell population** ‚Äî T cells (clonotypes, antigen-specific), endothelial cells,
  pericytes, astrocytes, microglia, neurons, tanycytes, muscle/satellite stem cells, immune
  (CD45+) / tumor / stromal cells, iPSC/PSC, RPE.
- **Cell-surface marker / antigen** ‚Äî CD45, CD3, CD4, CD45RA, CCR7, CXCR5 (define populations).
- **Tissue / anatomical structure** ‚Äî brain (microregions, cortex), lung, intestine/colon,
  dorsal root ganglion, tumor, embryo, retina.
- **Organoid / 3D culture** ‚Äî cerebral, cortical, intestinal, colon, ovarian/tumor, retinal.
- **Small molecule / compound / drug** ‚Äî screening libraries, vanillin, reagents (proteinase K,
  TRI Reagent, lysozyme, paraformaldehyde, Triton X-100, MSTFA), fluorophores.
- **Metabolite / lipid** ‚Äî fatty acids, triacylglycerol, untargeted-metabolomics features.
- **Phenotype / readout** ‚Äî viability, DNA-damage foci, electrophysiological properties, cytokine
  production, solubility.

**Procedural / methods entities (distinctive to this source class):**
- **Protocol / step / substep** ‚Äî the ordered procedure itself (often versioned, DOI'd).
- **Reagent / kit / buffer / consumable** ‚Äî with concentrations, ratios, catalog identity.
- **Instrument / platform** ‚Äî MinION, Illumina MiSeq/HiSeq, flow cytometer, patch-clamp rig,
  MEA, confocal microscope, LC-MS/GC-MS, NMR, micromanipulator.
- **Assay / technique** ‚Äî PCR, RT-qPCR, RT-LAMP, SDS-PAGE/western, IP, IHC/IF, FACS,
  scRNA-seq/CITE-seq, ATAC-seq, Hi-C, ChIP-seq, patch-clamp, mass spec.
- **Sample / specimen** ‚Äî swab, lysate, single-cell suspension, nuclei, tissue section, library.
- **Library / sequencing product / data file** ‚Äî amplicon library, FASTQ reads, consensus genome,
  count matrix, spectra.
- **Parameter / condition** ‚Äî temperature, OD600, incubation time, dose, A260/A280, fluorophore panel.
- **Software / pipeline / package** ‚Äî GATK/Mutect2, Scanpy, Pangolin, Pyteomics, edgeR, Limix.
- **Author / lab / consortium** ‚Äî lab-namespaced protocols (Endy, Sauer, Smolke, Wittrup;
  Earth Microbiome Project; iGEM teams) ‚Äî a strong **provenance/attribution** signal.
- **Control / standard** ‚Äî isotype controls, FMO, WT-vs-KO clonal pairs, reference primers.

### Recurring RELATION (edge) types

**Procedural / transformation edges (the backbone of this source class):**
- **protocol ‚Üí has_step ‚Üí step** (and **step ‚Üí precedes ‚Üí step**: ordered workflow / sequence).
- **reagent / kit ‚Üí applied_to ‚Üí sample** (often with quantitative ratio, e.g. "2 mL buffer per 10 mL culture").
- **sample ‚Üí transformed_into ‚Üí product** (cells ‚Üí lysate ‚Üí DNA/RNA ‚Üí library ‚Üí reads ‚Üí genome).
- **assay / instrument ‚Üí measures / detects / quantifies ‚Üí entity** (RT-qPCR detects pathogen RNA;
  cytometer quantifies marker+ population; patch-clamp records neuron activity).
- **antibody / primer / probe ‚Üí targets / binds ‚Üí antigen / gene region** (specificity edge).
- **enzyme ‚Üí acts_on ‚Üí substrate / DNA site** (Cas9 cuts locus; Cre recombines LoxP; Tn5 inserts into open chromatin).
- **parameter ‚Üí affects / optimizes ‚Üí outcome** (Œ≤-ME increases transformation efficiency;
  media choice affects yield; dose affects viability) ‚Äî a **conditional/quantitative** edge.
- **protocol ‚Üí uses ‚Üí instrument / software / reagent** (resource dependency).
- **protocol ‚Üí produces ‚Üí data_product** (library, count matrix, spectrum, structure).

**Provenance / organizational edges (strong in lab notebooks & versioned protocols):**
- **protocol / notebook_entry ‚Üí authored_by ‚Üí person / lab / consortium.**
- **protocol ‚Üí has_version ‚Üí version** (protocols.io v1/v2; DOI per version).
- **experiment ‚Üí references / derived_from ‚Üí prior_experiment** (Open Notebook Science:
  Exp098 ‚Üí Exp064 ‚Äî explicit provenance chain).
- **protocol ‚Üí variant_of / adapts ‚Üí another_protocol** (e.g. "modified version of Illumina").
- **protocol ‚Üí conforms_to ‚Üí standard** (Earth Microbiome 16S; ARTIC).
- **protocol ‚Üí bundled_in ‚Üí collection** (OmniPrep collection; iGEM team protocol sets).

**Biological/contextual edges embedded in procedures (overlap with article KG):**
- **marker ‚Üí defines / identifies ‚Üí cell_type** (CD45+ ‚Üí immune cell; surface-panel ‚Üí population).
- **cell_type ‚Üí isolated_from / resides_in ‚Üí tissue / anatomical_region.**
- **stem_cell ‚Üí differentiates_into ‚Üí cell_type / organoid** (iPSC ‚Üí RPE; PSC ‚Üí cerebral organoid).
- **perturbation (CRISPR KO/KD, drug) ‚Üí causes / induces ‚Üí phenotype / expression_change**
  (the closest thing here to an article-style causal edge ‚Äî and the most KG-valuable).
- **pathogen ‚Üí infects ‚Üí host_cell / tissue** (M. tuberculosis ‚Üí lung cells; SARS-CoV-2 ‚Üí swab specimen).
- **gene / transcript ‚Üí expressed_in / located_at ‚Üí cell / spatial_coordinate** (single-cell & spatial).
- **compound ‚Üí has_property ‚Üí value** (vanillin solubility in methanol; metabolite abundance).

### Implications for the BioOKF type universe

1. **A dual node universe is needed.** Article-derived KGs are dominated by biological entities;
   protocol/notebook sources add a parallel **methods/provenance** layer (Protocol, Step, Reagent,
   Instrument, Assay, Sample, DataProduct, Lab/Author, Version, Standard). Modeling these makes the
   KG *method-aware and reproducible*, and lets evidence be traced to "how it was generated."
2. **Procedural edges (precedes, transforms_into, applied_to, measures, produces) are first-class**
   and largely absent from article-claim extraction ‚Äî they are the defining signal of this class.
3. **Quantitative/conditional edges** (parameter‚Üíaffects‚Üíoutcome, reagent‚Üíratio‚Üísample) carry
   numeric attributes; BioOKF edges should support typed quantitative properties (concentration,
   temperature, dose, time, OD, A260/A280).
4. **Provenance edges are unusually rich** here (versioned DOIs, lab-namespaced protocols,
   experiment‚Üíprior-experiment links in Open Notebook Science). This source class is the natural
   home for the KG's **attribution, versioning, and reproducibility** subgraph.
5. **The bridge to the biological ontology** is the perturbation‚Üíphenotype and
   marker‚Üícell_type‚Üítissue‚Üíorganism chains embedded in protocols. These overlap directly with
   article-derived entities (genes, cell types, drugs, organisms, variants), so protocol nodes
   can be linked into the same biomedical entity space rather than living in isolation.
