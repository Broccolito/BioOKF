## Entity & relation patterns observed

This source class (clinical-specialty peer-reviewed literature) is dominated by **intervention →
outcome** claims about **named diseases and procedures in specific patient populations**, with
quantitative comparisons between treatment options. The recurring patterns below should seed the
BioOKF type universe.

### Observed entity (node) types

- **Disease / Condition** — diagnosed clinical entity (knee/hip osteoarthritis, spondylolisthesis,
  rotator cuff tear, osteosarcoma, heart failure [HFrEF/HFpEF], aortic stenosis, NSCLC, breast cancer,
  multiple myeloma, ischemic stroke, multiple sclerosis, Parkinson's, epilepsy, Alzheimer's, RA, SLE,
  HIV, tuberculosis, COVID-19, periprosthetic joint infection). Often carries subtype/stage modifiers.
- **Injury / Fracture** — trauma-specific (femoral neck fracture, distal radius fracture, lung/thoracic
  injury), with displacement/grade attributes; distinguished from chronic disease.
- **Anatomical structure** — bone, joint, ligament, tendon, nerve (median nerve), brain nucleus (STN),
  spinal level (T6–T11), vessel; the site that procedures act on or tumors occupy.
- **Procedure / Surgery / Technique** — arthroplasty (TKA/THA/TSA), fusion, decompression, ORIF,
  arthroscopic repair, limb salvage, PCI, TAVR, thrombectomy, carpal tunnel release, hernia repair;
  often paired as "A vs B" comparators.
- **Implant / Device / Graft / Mesh** — prosthesis designs, growing rods, DBS leads, biodegradable
  drains, valve devices, allograft/autograft (HT/BTB/QT), surgical mesh; the manufactured/biological
  object placed in the patient.
- **Drug / Therapeutic agent** — small molecules and biologics (SGLT2 inhibitors, anticoagulants,
  pembrolizumab, trastuzumab, CDK4/6 inhibitors, tenecteplase, ocrelizumab, lecanemab, JAK inhibitors,
  belimumab, anifrolumab, remdesivir, isoniazid, antibiotics, ART, temozolomide).
- **Drug class / Mechanism target** — receptor or pathway the agent acts on (PD-L1, HER2, BAFF,
  IFNAR, CD20, BCMA, amyloid-β, JAK/STAT, 3CL protease); a frequent bridge node between drug and effect.
- **Biomarker / Measurement** — PD-L1 tumor proportion score, ejection fraction, Harris Hip Score,
  KCCQ, mRS, radiographic measures, INR; both eligibility criteria and outcome variables.
- **Outcome / Endpoint** — functional scores, survival, recurrence, revision, hospitalization,
  reoperation, complication, return-to-sport, ARIA, IRIS; the dependent variable of a claim.
- **Complication / Adverse event** — anastomotic leak, infection, aseptic loosening, ARIA, AVN,
  bleeding; an entity that connects procedure/drug to a negative outcome.
- **Pathogen** — SARS-CoV-2, M. tuberculosis, HIV, biofilm-forming bacteria in PJI.
- **Cell type** — B cell, T cell (engineered CAR-T), tumor cell.
- **Population / Cohort** — defined by age band, risk stratum, prior-therapy status, comorbidity;
  modifies the validity scope of nearly every claim.
- **Study / Trial / Evidence artifact** — named RCTs (KEYNOTE, EMPEROR, PARTNER, ORATORIO, NOR-TEST,
  C-seal, CHIEF-HF), meta-analyses, registries (NJR, AJRR), guidelines (AAOS, ESC, WSES-AAST),
  case reports; the provenance node carrying evidence level and effect size.
- **Imaging modality / finding** — MRI, CT, PET, radiograph, CMR; both a diagnostic method and a
  source of measured findings.

### Observed relation (edge) types

- **treats / indicated-for** — Drug|Procedure → Disease|Injury (the core therapeutic claim).
- **prevents** — Drug → Disease (e.g., isoniazid → TB; C-seal device → anastomotic leak).
- **improves / reduces / increases** — Intervention → Outcome|Biomarker (quantified effect direction).
- **causes / associated-with / complicated-by** — Procedure|Drug → Complication; Disease → Disease
  (comorbidity, e.g., AF complicating STEMI).
- **compared-with / non-inferior-to / superior-to** — Intervention ↔ Intervention (head-to-head, the
  most frequent edge in this class; carries an effect-size/p-value attribute).
- **targets / inhibits / binds** — Drug → Molecular target|Pathway|Antigen (mechanism edge).
- **predicts / risk-factor-for** — Patient factor|Biomarker → Outcome|Complication.
- **performed-on / located-in / acts-on** — Procedure|Tumor → Anatomical structure.
- **delivered-via / fixed-by / augmented-with** — Procedure → Device|Graft|Mesh.
- **measured-by / diagnosed-by / guided-by** — Outcome → Biomarker|Imaging; Procedure → Imaging.
- **expressed-on** — Antigen|Biomarker → Cell type (e.g., BCMA on MM cell, PD-L1 on tumor).
- **recommends** — Guideline → Drug|Procedure (with evidence grade).
- **studied-by / reported-in / evidenced-by** — Disease|Intervention → Study|Trial|Registry
  (provenance edge carrying evidence level, sample size, follow-up).
- **stratified-by / restricted-to** — Claim → Population|Subtype (scopes the validity of an edge).
- **followed-by / staged-with / timing-relative-to** — Procedure|Drug → Procedure|Drug (sequence,
  e.g., thrombectomy → adjunctive tenecteplase; ART timing relative to TB treatment).

### Modeling implications for BioOKF

1. **Comparative, population-scoped claims dominate.** The atomic fact is rarely "Drug treats Disease"
   but "Intervention A vs B → effect on Outcome, in Population P, per Study S." Edges need first-class
   qualifiers for **population/subtype**, **effect direction + magnitude**, and **evidence
   provenance** — otherwise contradictory specialty findings cannot be reconciled.
2. **Procedure/Device/Implant/Graft are distinct node types**, not collapsible into "treatment."
   Orthopaedics especially links Procedure → Implant → revision Outcome, and registries make
   Implant-level device survival a primary entity.
3. **A mechanistic bridge layer (Drug → Target/Pathway → Cell/Process → Disease)** recurs across
   oncology, immunology, and neurology and connects molecular and clinical layers.
4. **Complication and Outcome deserve separate types** — a complication can be both an effect of one
   edge and a cause/predictor in another.
5. **Evidence artifacts (Trial/Guideline/Registry/Case report) should be reifiable nodes** so each
   clinical edge can attach evidence level, sample size, follow-up duration, and effect size.
