## Description

Contains data that was used for the shiny app.

*   pairwise\_comparisons.zip:

Contains all pairwise comparisons of clusters using the RaceID function diffexpnb.

 

*   shiny\_\_sc\_ndata\_\_pairwise\_comparisons\_of\_all\_clusters\_Human\_Liver\_Atlas\_Aizarani\_et\_al\_Nature\_2019.csv.gz:
