# Genotoxicity

Genotoxicity is the process by which microbiota-derived agents (colibactin, secondary bile acids,
reactive oxygen species) damage host DNA in colonic epithelial cells, producing double-strand
breaks and mutational signatures that seed colorectal cancer driver mutations. Limiting oxidative
DNA damage reduces microbe-induced colitis-associated CRC (Irrazabal et al., Nat. Commun. 2020, as
reviewed by Janney et al., Nature 2020).
