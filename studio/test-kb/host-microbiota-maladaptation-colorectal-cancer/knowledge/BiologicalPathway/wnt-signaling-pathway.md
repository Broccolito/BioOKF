# Wnt signaling pathway

The Wnt/β-catenin signalling pathway controls intestinal stem-cell proliferation and is a
canonical driver of colorectal carcinogenesis. Microbiota can hijack this pathway: *F. nucleatum*
FadA activates Wnt via E-cadherin, while butyrate-producing probiotics such as *Clostridium
butyricum* modulate Wnt to suppress tumour development (as reviewed by Janney et al., Nature 2020).
