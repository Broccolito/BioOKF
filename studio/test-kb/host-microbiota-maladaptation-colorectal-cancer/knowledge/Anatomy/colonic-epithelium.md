# Colonic epithelium

The single-cell layer of the intestinal epithelium that forms the barrier between the luminal
microbiota and host tissue. Its disruption is a central theme in microbiota-driven colorectal
carcinogenesis; the mucus layer overlying the colonic epithelium can be degraded by mucus-foraging
bacteria when dietary fiber is scarce (Janney et al., Nature 2020).
