# FadA adhesin

FadA is a surface adhesin of *Fusobacterium nucleatum* that binds host E-cadherin on colonic
epithelial cells. This engagement activates Wnt/β-catenin signalling and promotes proliferation of
colorectal cancer cells, contributing to *F. nucleatum*-driven tumorigenesis (as reviewed by
Janney et al., Nature 2020).
