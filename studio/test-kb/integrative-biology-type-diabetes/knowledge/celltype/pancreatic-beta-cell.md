# Pancreatic beta-cell

Secretes [insulin](../molecule/insulin.md) in response to glucose. Compensatory
hyperinsulinaemia promotes hepatic de novo lipogenesis, steatosis and WAT expansion; beta-cell
function then declines (partly glucolipotoxicity) just before T2D onset, with population-specific
differences (faster decline in African Americans).
