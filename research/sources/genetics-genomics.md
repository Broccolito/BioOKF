## Entity & relation patterns observed

This source class is dominated by **variant-centric** knowledge. The recurring
biomedical node types and the edge types that connect them:

### Entity (node) types that recur
- **Genetic variant** — SNP/SNV, indel, structural variant (deletion,
  duplication, inversion, CNV), VNTR/tandem repeat; identified by rsID,
  chrom:pos:ref:alt, or HGVS. The single most common entity.
- **Gene** — protein-coding gene, pseudogene, lncRNA gene; the unit that variants
  map to and that carries constraint scores (pLI, LOEUF).
- **Genomic region / locus** — GWAS locus, haplotype block, MHC region, TAD,
  cCRE/enhancer/promoter, CpG site, regulatory interval, chromosome/centromere.
- **Allele / genotype** — including HLA alleles and star alleles (CYP2D6*N),
  diplotypes, copy-number states.
- **Trait / phenotype** — disease (T2D, schizophrenia, Alzheimer's, CAD, cancer
  types), quantitative trait (height, BMI), endophenotype, drug-response/
  metabolizer phenotype, biomarker level.
- **Molecular quantitative phenotype** — gene-expression level (eQTL target),
  splicing event (sQTL), DNA-methylation level (mQTL/CpG), protein level (pQTL),
  microbial taxon abundance.
- **Tissue / cell type** — the context in which a regulatory effect is active
  (49 GTEx tissues, single-cell clusters).
- **Population / ancestry group / cohort** — continental ancestry, biobank
  cohort, ancient population; carries allele-frequency context.
- **Sample / individual / donor** — the observation unit (patients, trios,
  postmortem donors).
- **Drug / chemical** — pharmacogenomic substrates linked to metabolizing genes.
- **Pathway / gene set / protein complex** — enrichment targets of GWAS loci.
- **Polygenic score (PRS/PGS)** — an aggregate entity over many variants.
- **Database / study / resource** — provenance nodes (GWAS Catalog, ClinVar,
  gnomAD, dbSNP, GTEx, DepMap) that hold the assertions.

### Relationship (edge) types that recur
- **variant — associated-with → trait/disease** (GWAS association; the core edge,
  with effect size, p-value, ancestry as edge attributes).
- **variant — located-in / maps-to → gene / region** (genomic annotation).
- **variant — regulates → gene expression / splicing / methylation in tissue**
  (eQTL / sQTL / mQTL; carries tissue and cell-type context).
- **variant/gene — causes → Mendelian disease** (rare-variant causation, with
  inheritance mode and pathogenicity classification).
- **gene — has-constraint → score** (pLoF tolerance, LOEUF/pLI).
- **variant/gene — has-allele-frequency-in → population** (population genetics).
- **gene/variant — driver-of → cancer type** (somatic-mutation driver edge).
- **allele/genotype — affects → drug response / metabolizer phenotype**
  (pharmacogenomic gene–drug edge).
- **HLA allele — confers-risk-for → autoimmune disease** (MHC association).
- **exposure — causally-affects → outcome** via genetic instrument (Mendelian
  randomization; a variant-anchored causal edge between two traits).
- **enhancer — contacts/regulates → promoter/gene** (3D-genome / chromatin loop).
- **host variant — modulates → microbial taxon abundance** (host-microbiome).
- **gene — is-essential-in / dependency-of → cell line / cancer context**
  (CRISPR functional-genomics edge).
- **polygenic score — predicts → disease risk** (aggregate-to-phenotype edge).
- **population — admixed-from / migrated-from → ancestral population**
  (paleogenomics demographic edges).
- **variant/assertion — reported-in / curated-by → study/database** (provenance,
  with evidence level and review status).

### Implications for the BioOKF type universe
1. The graph must support **first-class variant nodes** with rich identifiers
   (rsID, HGVS, chrom-pos) and frequency/effect attributes — variants are the hub
   connecting nearly every other entity type.
2. Edges are heavily **context-qualified**: the same variant–gene edge means
   different things depending on **tissue, cell type, ancestry, and direction
   (cis/trans)**. BioOKF edges likely need qualifier slots (tissue, population,
   effect size, p-value, evidence/provenance).
3. A **molecular-QTL layer** (eQTL/sQTL/mQTL/pQTL) is the connective tissue
   between non-coding variants and genes — distinct from direct coding-consequence
   edges. Modeling "variant → molecular phenotype → trait" as a two-hop path is
   essential for mechanism (TWAS, colocalization, MR all traverse it).
4. **Provenance/evidence is itself structured** (study, database, ancestry,
   review status, effect statistics). Curated resources (GWAS Catalog, ClinVar,
   gnomAD) are reusable provenance anchors rather than mere citations.
5. **Causal vs. associational** edges should be distinguishable — GWAS yields
   associations, while MR and fine-mapping/colocalization upgrade selected edges
   to causal claims; the type system benefits from an edge-confidence/causality
   dimension.
6. **Aggregate entities** (polygenic scores, gene-set/pathway enrichment, gene
   constraint) sit above the per-variant layer and need their own node types and
   "composed-of / summarizes" edges back to variants and genes.
