## Entity & relation patterns observed

This source class (tweetorials + expert blogs) is unusually **mechanism-dense
and causal-chain-heavy**. Because the authors are teaching, they make the
intermediate steps of a relationship explicit ("X causes Y *because* Z does W"),
which is exactly the multi-hop structure a knowledge graph wants ‚Äî but they
encode it in prose, with claims that are hedged, dated, and sometimes corrected
in follow-up posts.

### Observed entity (node) types

- **Disease / clinical syndrome** ‚Äî pulmonary embolism, hyponatremia, DKA, heart
  failure, sepsis, prion disease, COVID-19; the explanandum most threads orbit.
- **Gene** ‚Äî PRNP, HTT, SOD1, SMN1/2, APOB, APOC3, MAPT, C9orf72; named as the
  causal locus or therapeutic target.
- **RNA species / transcript / isoform** ‚Äî mRNA, pre-mRNA, spliced/unspliced,
  miRNA (miR-122), splice variants; first-class in the RNA/ASO/splicing sources.
- **Protein** ‚Äî PrP, hemoglobin, RNase H1, spike protein, cytokines, enzymes;
  the functional intermediary between gene and phenotype.
- **Variant / mutation** ‚Äî PRNP D178N, spike mutations, GWAS loci, ESBL/AmpC
  resistance alleles; carries a specific genotype‚Üíphenotype edge.
- **Drug / therapeutic modality** ‚Äî ASOs (nusinersen, ION717), small molecules,
  vitamin cocktails, senolytics, antibiotics; the intervention node.
- **Drug target / receptor / pathway** ‚Äî receptors, RNase H pathway, complement,
  V/Q physiology; the mechanism anchor.
- **Cell type** ‚Äî T cells, B cells, mast cells, senescent cells, tumor clones;
  often the site where a mechanism plays out.
- **Physiological process / parameter** ‚Äî V/Q ratio, preload/afterload, anion
  gap, serum osmolality, perfusion, methylation level; quantitative mediators.
- **Pathogen / microbe / virus** ‚Äî SARS-CoV-2, gram-negative bacteria, phages,
  prions; subject of virology/micro sources.
- **Biomarker / lab value** ‚Äî lactate, sodium, ferritin, B12, DNAm age, CVP;
  the measured signal a thread interprets.
- **Anatomical structure / organ** ‚Äî lung, right ventricle, kidney, brain,
  telomere/chromosome end; localizes the claim.
- **Method / assay / technology** ‚Äî scRNA-seq, IVUS/OCT, AlphaFold, nanopore,
  GWAS, methylation clock; both a tool node and a provenance qualifier.
- **Clinical trial / regulatory milestone** ‚Äî IND, ION717 trial, KEYNOTE-style
  named studies; reifiable evidence node.
- **Author / community / hashtag** ‚Äî @tony_breu, #NephMadness, #IDtwitter; a
  provenance/credibility node distinctive to this social class.

### Observed relation (edge) types

- **causes / leads-to (multi-hop mechanism)** ‚Äî the signature edge: Disease ‚Üí
  Process ‚Üí Molecule ‚Üí Symptom, with every hop stated (PE‚ÜíV/Q-mismatch‚Üíhypoxia;
  iron-deficiency‚Üíenzyme-deficit‚Üímitochondrial-dysfunction‚Üífatigue).
- **treats / lowers / modulates** ‚Äî Drug ‚Üí Target/RNA/Protein ‚Üí Disease (ASO ‚Üí
  PRNP mRNA cleavage ‚Üí PrP lowering ‚Üí prion-disease delay).
- **inhibits / cleaves / binds / degrades** ‚Äî Drug|Enzyme ‚Üí molecular target
  (RNase H1 cleaves RNA; senolytic clears senescent cell).
- **regulates / splices / blocks-translation** ‚Äî RNA-feature|ASO ‚Üí isoform/protein
  output (splice-modulating ASO ‚Üí exon inclusion ‚Üí protein isoform).
- **associated-with / risk-factor-for** ‚Äî Variant|Biomarker ‚Üí Trait|Outcome (GWAS
  variant ‚Üí trait; long telomeres ‚Üî cancer risk, contested).
- **measured-by / interpreted-as / diagnosed-by** ‚Äî Biomarker|Lab ‚Üí State (lactate
  ‚Üí hypoperfusion; serum osmolality ‚Üí hyponatremia subtype).
- **expressed-in / acts-in / localized-to** ‚Äî Gene|Process ‚Üí Cell|Organ (mechanism
  sited in RV, kidney, T cell).
- **resistant-to / stable-against** ‚Äî Pathogen ‚Üí Drug (AmpC ‚Üí cephalosporins;
  cefepime stable against AmpC) ‚Äî a polarity-bearing antibiotic edge.
- **tested-in / studied-by / reported-in** ‚Äî Drug|Hypothesis ‚Üí Trial|Paper (ION717
  trial; "ASOs work" evidence), the provenance edge.
- **compared-with / non-inferior / null-effect** ‚Äî Intervention ‚Üî Intervention or
  Intervention ‚Üí Outcome with a *negative* result (vitamin-C cocktail ‚Üí no
  mortality benefit) ‚Äî these sources are notably willing to assert null edges.
- **authored-by / shared-via / tagged** ‚Äî Claim ‚Üí Author|Hashtag|Platform, a
  social-provenance edge unique to this class.

### Modeling implications for BioOKF

1. **Reified multi-hop causal chains are the native unit.** Unlike a database
   row, a tweetorial asserts a *chain* (A‚ÜíB‚ÜíC‚ÜíD). BioOKF should ingest these as
   linked binary edges that share intermediate nodes, preserving the path so the
   "because" is queryable ‚Äî collapsing to A‚ÜíD loses the teaching content.
2. **Edge polarity and effect direction must be first-class** (lowers vs raises,
   stable-against vs resistant-to, null vs positive). This class frequently
   asserts *negative/null* relationships, which are valuable and easy to lose.
3. **Provenance/credibility is weaker and must be modeled explicitly.** A
   tweetorial or blog post is expert-but-not-peer-reviewed; edges should carry a
   source-type qualifier (tweetorial | lab-blog | society-blog) and author
   identity, and ideally a "contested / later-corrected" flag (several CureFFI
   and Mitteldorf posts revise earlier claims).
4. **RNA species and isoforms need to be distinct node types**, not folded into
   "gene" ‚Äî the ASO/splicing sources only make sense if pre-mRNA, mature mRNA,
   splice isoform, and protein are separate nodes connected by process edges.
5. **Method/technology nodes double as provenance qualifiers** (scRNA-seq,
   AlphaFold, GWAS): the same entity is both "what was used" and "how confident
   the edge is," arguing for method as a reifiable attribute on edges.
6. **Cross-platform identity resolution matters.** The same thread appears as an
   x.com permalink and a threadreaderapp unroll; the same author blogs and
   tweets. BioOKF should treat author+claim as the dedup key, not URL.
