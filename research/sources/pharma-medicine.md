# Pharmacology & Pharmaceutical Medicine — Source Catalog

A catalog of real, openly-accessible peer-reviewed pharmacology and pharmaceutical
medicine sources (drug mechanism, pharmacokinetics, drug–target interaction, clinical
pharmacology, drug repurposing, pharmacogenomics, pharmacovigilance, and the major
open drug knowledgebases). Compiled for the BioOKF type universe: each row notes the
biomedical **entities** and **relationships** the source surfaces.

All URLs were located via web search and primary sources sampled/verified during
compilation (2026-06-25). PMC = PubMed Central open-access full text; NAR = Nucleic
Acids Research Database Issue (open access); GtoPdb = IUPHAR/BPS Guide to Pharmacology.

## Entity & Relation Patterns Observed

This source class (pharmacology and pharmaceutical medicine) is remarkably consistent

by its relationships (to targets, enzymes, transporters, diseases, genes, and other drugs).

### Recurring entity (node) types
- **Drug / compound / active pharmaceutical ingredient** — the anchor node (small
  molecule, prodrug, peptide, monoclonal antibody, biologic, nanoparticle formulation).
- **Drug target** — protein/receptor/enzyme/ion channel (e.g., GPCR, kinase, HMG-CoA
  reductase, factor Xa, PD-1, mu-opioid receptor, SERT, GLP-1 receptor).
- **Metabolizing enzyme** — predominantly CYP450 isoforms (CYP2C8, CYP1A2, CYP2C19, …).
- **Transporter** — efflux/uptake proteins (P-glycoprotein/ABCB1, ABC/RND/MFS pumps).
- **Gene / genetic variant** — pharmacogenomic markers, metabolizer-status alleles (TPMT, CYP variants).
- **Disease / indication / condition** — the therapeutic endpoint.
- **Adverse event / side effect (ADR)** — MedDRA-style outcome terms.
- **Metabolite** — active or inactive product of biotransformation.
- **Pathway / biological process / signaling network** — mechanistic context.
- **Pharmacologic / drug class** — grouping nodes (statins, beta-blockers, SSRIs, DOACs, TKIs).
- **PK/PD parameter** — Ki, Kd, IC50, EC50, Emax, clearance, half-life, Vd, AUC, Cmax
  (often attributes/quantified edges rather than standalone nodes).
- **Drug label / clinical guideline / regulatory artifact** — SPL sections, dosing guidelines.
- **Cell type / tissue / organ / population** — context for PK and PD (hepatic/renal impairment, pediatric).
- **Pathogen / virus / bacterium** — target organism for anti-infectives.

### Recurring relationship (edge) types
- **drug — targets / inhibits / activates / binds — target** (with affinity/kinetics: Ki, Kd, kon/koff; agonist/antagonist/allosteric modifier).
- **drug — has mechanism of action — (pathway / target effect)**.
- **drug — is substrate of / inhibits / induces — enzyme (CYP)** and **— transporter (P-gp)**.
- **drug — metabolized to — metabolite**; **metabolite — has activity on — target**.
- **drug — interacts with — drug** (drug–drug interaction; metabolic, transporter-mediated, pharmacodynamic).
- **drug — interacts with — food** (food–drug interaction).
- **drug — treats / indicated for — disease**; **drug — repurposed for — new disease**.
- **drug — causes — adverse event/side effect** (with frequency; disproportionality signal: ROR/PRR/BCPNN/EBGM).
- **gene / variant — affects response/metabolism of — drug** (pharmacogenomic association, with evidence level).
- **drug — has PK parameter — value** (clearance, half-life, Vd, AUC, exposure).
- **dose — produces — exposure — produces — response/effect** (the dose–exposure–response chain, the spine of PK-PD/QSP).
- **drug — belongs to — pharmacologic class**; **class — shares — mechanism**.
- **target / pathway — implicated in — disease** (links the drug-target edge to the disease edge).
- **antibody — binds — antigen/epitope**; **antibody — recruits — effector mechanism (ADCC/CDC)**.
- **antimicrobial — inhibited by — resistance mechanism** (target alteration, efflux, enzymatic inactivation).
- **side effect — associated with — biological process** (process–drug–side-effect networks).

### Format & access patterns
- Dominant format is **open-access HTML full text on PMC** (peer-reviewed reviews,
  case studies, and database descriptors), plus **NAR Database-Issue HTML** for the
  major knowledgebases (DrugBank, DrugCentral, ChEMBL, GtoPdb, SIDER).
- A distinct **structured/bulk-data tier** exists and is highly extractable: openFDA
  (JSON via Elasticsearch API + zipped bulk downloads), DailyMed SPL (XML/JSON/zip),
  PharmGKB (CC-licensed downloadable annotations), STITCH/SIDER, ChEMBL — these provide
  pre-built drug–target, drug–ADR, drug–gene, and drug–label triples ready for ingestion.
- Notably the structured drug knowledgebases are themselves already **graph-shaped**
  (DrugBank, DrugCentral, ChEMBL, GtoPdb, PharmGKB, SIDER/STITCH), making them
  near-direct templates for BioOKF drug-centric edge types and identifier cross-mapping.
